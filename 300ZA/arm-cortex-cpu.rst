ARM Cortex CPU
===============

.. contents:: Contents
    :local:

The OpenIMU300ZA uses ST's powerful Cortex M4 series of Microcontrollers.

* FPU
* DSP instructions
* 1MByte Flash
* 192KB SRAM
* 168 MHz
* Rich Set of peripherals


Learn more about OpenIMU300ZA's CPU at http://www.st.com/en/microcontrollers/stm32f405rg.html.
